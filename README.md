Prerequisities
- Node installed (v12+)
- NPM/Yarn installed
- (optional) Quasar installed https://quasar.dev/quasar-cli/installation

## Install the dependencies
```bash
yarn
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
// or
yarn start
```

### Lint the files
```bash
yarn run lint
```
